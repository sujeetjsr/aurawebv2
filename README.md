# Mega Bundle App (Selection Mode)

## How to Use
1.  **Open the App**: Double-click `index.html`.
2.  **Select Bundles**: 
    -   **Zoom**: Tap any image to see it full screen. Tap anywhere to close.
    -   **Select**: Tap the **Green "SELECT üëÜ"** button. 
    -   **Animation**: Watch the image **fly** into the progress bar at the bottom!
    -   **Progress Bar**: A green bar fills up (10% per item) as you select more.
    -   **Remove**: The button turns **Red "REMOVE ‚ùå"** if you change your mind.
3.  **Completion**: Once the bar hits 100% (10 items), the "BUY NOW" button activates with a celebration vibration.
4.  **Buy Now**: Click the Green "BUY NOW" button to order via WhatsApp.

## Configuration
-   **Add Photos**: Add images to `photo/` folder and update `script.js`.
-   **Change WhatsApp**: Edit `whatsappNumber` in `script.js`.

## Features
-   **Visual Progress**: Satisfying "fill the bar" mechanic.
-   **Immersive Animations**: Flying images, pulsing counters, and smooth transitions.
-   **Mobile-First**: Designed for touch interaction with clear, large buttons.
-   **App-Like Feel**: Fixed bottom bar, modal overlays, and haptic feedback.
